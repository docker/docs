If you have an organization account and you would like to remove a team member followÂ these steps.

Docker Hub

For Docker Hub follow these steps

  1. Log in to your Docker Hub account.
  2. Click Organizations in the top navigation bar.
  3. Select the organization that includes the team member you would like to remove.
  4. Select Teams in the secondary navigation bar.
  5. Select the team that include the team member you would like to remove.
  6. Click the red X next to the team member you would like to remove.

Docker Cloud

For Docker Cloud follow these steps:

  1. Log in to your Docker Cloud account.
  2. Click on the user icon menu at the top right.
  3. Select the organization that includes the team member you would like to remove.
  4. Select Teams in the top navigation bar.
  5. Select the team that include the team member you would like to remove.
  6. Hover over the username you would like to remove.
  7. Click the grey X that appears when you are hovering.
