The Docker Cloud Agent is supported and has been tested on the following Linux distros:

  • Ubuntu 14.04, 15.04, 15.10
  • CentOS 7
  • Debian 8
  • Fedora 21, 22, 23
  • Red Hat Enterprise Linux 7
