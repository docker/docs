--- title: "What should I do if a Docker Cloud object gets stuck in the delete state?" id: 1154 draftstate: inactive deleted: false layout: docs permalink: /kb/1154/ source: https://success.docker.com/@api/deki/pages/1154/contents tags: - tag: "article:topic" - tag: "product:cloud" - tag: "stage:reviewed" --- {% raw %}

When trying to delete a Docker Cloud object, if the action will not complete (gets stuck in the delete state), open a support case with the following information:

  • Your Docker Cloud username and/or organization
  • The object that is stuck during delete
