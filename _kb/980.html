Question

Where is the nodes/services monitoring tab in Docker Cloud?

Answer

There is currently no node monitoring interface in Docker Cloud. A new monitoring system is being developed that leverages the monitoring being done by the Docker Engine, as well as better integration with monitoring service providers. A number of 3rd party monitoring solutions are also available now.
