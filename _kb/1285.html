--- title: "How do I provide my license file during the UCP command line installation?" id: 1285 draftstate: inactive deleted: false layout: docs permalink: /kb/1285/ source: https://success.docker.com/@api/deki/pages/1285/contents tags: - tag: "article:howto" - tag: "component:ucp" - tag: "license_automation" - tag: "product:datacenter" - tag: "stage:reviewed" - tag: "ucp" --- {% raw %}

Go to Docker Store and download your UCP license if you don't already have the file.

When installing UCP, use the -v option to create a volume for the license file (shown in red): 

docker run --rm -it \
  --name ucp \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v /path/to/my/docker_subscription.lic:/docker_subscription.lic \
  docker/ucp \
  install [command options]
{% endraw %}
