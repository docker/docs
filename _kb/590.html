Question

  • Is Docker Trusted Registry (DTR) supported in a high-availability (HA) configuration?
  • Can I load balance between instances of DTR to provide HA capabilities?

Answer

Yes. The latest version of DTR allows for high-availability and load balancers. Refer to Docker Reference Architecture: Docker Datacenter Best Practices and Design Considerations for details.
