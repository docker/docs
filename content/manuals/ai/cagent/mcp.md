**MCP toolsets:**
- `command`: Command to run for local MCP servers
- `args`: Arguments for the command
- `ref`: Reference name for the MCP server
- `remote`: Configuration for remote MCP servers
  - `url`: Remote server URL
  - `transport_type`: Transport type (`sse` or `http`)
  - `headers`: HTTP headers for authentication
